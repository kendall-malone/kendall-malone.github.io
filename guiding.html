Clear Creek

Rapids on Clear Creek!

  • Phoenix Holes - Class IV
    • Named after the Phoenix Gold Mines. One of the first gold mines in Colorado and is supposedly haunted - spooky!
  • Deliverance - Class IV
  • Slam Dancer - Class IV(My favorote - drop 35 feet in a 40 foot span!)
  • Mountain Lion - Class III
    • Scott Lancaster was the first person reported to die by a mountain lion attack in Colorado. There is the Scott Lancaster Memorial Bridge crossing over this part of Clear Creek with Mountain Lion rapid right underneath.
  • Beaver I & II - Class II/III
  • Double Knife - Class IV+
  • Hell's Half Mile - Class IV+
  • Horrendous Left Hand Turn - Class IV
  • Corkscrew - Class III/IV
